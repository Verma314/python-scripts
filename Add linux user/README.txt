Create new user accounts on a Linux system and assign a specific group to the user account. 

The script takes the following inputs from a CSV file:

Preferred user name
The full name of the user
E-mail
User group

	
The input CSV file looks like as follow:

Albert,Albert Einstein,albert.einstein@ligo.org,Group-1
Richard.Richard Feynman,richard.feynman@icts.res.in,Group-2
